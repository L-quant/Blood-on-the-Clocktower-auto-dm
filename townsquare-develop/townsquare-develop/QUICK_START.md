# 快速启动指南

## 当前状态

系统已完成开发，但有一些ESLint警告需要处理。这些警告不影响功能，只是代码风格问题。

## 启动步骤

### 方法1: 忽略ESLint错误启动（推荐）

在`vue.config.js`中临时禁用ESLint：

```javascript
module.exports = {
  lintOnSave: false,  // 添加这一行
  // ... 其他配置
}
```

然后运行：
```bash
cd townsquare-develop
npm run serve
```

### 方法2: 修复ESLint错误后启动

运行自动修复：
```bash
cd townsquare-develop
npm run lint -- --fix
```

然后启动：
```bash
npm run serve
```

## 使用自动化系统

1. 打开浏览器访问 `http://localhost:8080`

2. 点击右上角的**齿轮图标**打开菜单

3. 点击**机器人图标**切换到"自动化说书人"标签

4. 点击**"打开控制面板"**

5. 在控制面板中：
   - 点击**"初始化自动化系统"**
   - 配置AI难度和自动化级别
   - 添加至少5名玩家（在Players标签中）
   - 返回自动化面板，点击**"启动"**

## 功能说明

### 控制面板功能

- **系统状态**: 显示当前系统状态（空闲/运行中/已暂停/错误）
- **游戏阶段**: 显示当前游戏阶段（准备/第一夜/白天/夜晚）
- **控制按钮**: 
  - 初始化：初始化自动化系统
  - 启动：开始自动化游戏
  - 暂停：暂停自动化
  - 恢复：恢复自动化
  - 停止：停止游戏

- **AI决策建议**: 显示待处理的AI决策（如果有）
- **配置选项**:
  - AI难度：简单/中等/困难
  - 自动化级别：手动/半自动/全自动
  - 调试模式：启用/禁用

- **错误显示**: 显示系统错误（如果有）
- **日志查看**: 调试模式下显示最近10条日志

## 常见问题

### Q: 为什么有ESLint错误？
A: 这些是代码风格警告，主要是未使用的参数。不影响功能，可以通过设置`lintOnSave: false`忽略。

### Q: 如何添加玩家？
A: 在菜单的"Players"标签中点击"Add"添加玩家。

### Q: 需要多少玩家才能开始？
A: 至少需要5名玩家才能开始游戏。

### Q: 自动化系统做什么？
A: 自动化系统会：
- 自动分配角色
- 自动处理夜间行动
- 自动管理投票流程
- 自动判断胜负
- 提供AI决策建议

### Q: 如何查看日志？
A: 在控制面板中启用"调试模式"，日志会显示在面板底部。
